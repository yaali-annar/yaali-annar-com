The noun cases are in form of postpositions.

Core cases:

| Case     | Morpheme |
| -------- | -------- |
| Ergative | ak       |
| Patient  | il       |
| Genitive | iš       |

Spatial cases:

| Case     | Morpheme |
| -------- | -------- |
| Locative | uk       |
| Allative | ar       |
| Ablative | tik      |

Other cases

| Case         | Morpheme |
| ------------ | -------- |
| Causative    | gat      |
| Benefactive  | sat      |
| Instrumental | os       |

### Pronouns

There's no number distinction in pronouns, just like in nouns.

| Pronoun    | Morpheme |
| ---------- | -------- |
| 1st Person | nik      |
| 2nd Person | kat      |
| 3rd Person | šus      |
